About Me

Hi! My name is Ikhwan and I love building things. My web development interest sparked back when I was 13 during the Harlem Shake meme era where I stumbled upon a piece of Javascript code that would make the elements on the page of the site that I was in to shake wildly. Another reason for my interest in web development is because I loved to mess around with 'inspect element' in the browser. I remember feeling like I was a hacker just because I could change the type for the element 'input' from 'password' to 'text' so that it would reveal the password in the input field.
